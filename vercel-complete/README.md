### Option 1: Vercel CLI
```bash
cd vercel-complete
npm install

```

### Option 2: GitHub + Dashboard
1. Copy folder ini ke repo GitHub baru
2. Push ke GitHub
3. Import di Vercel Dashboard
4. Set environment variables
